Run Instructions
*******************

- Make sure that the folder /Data/EmoMus5/Data/ contains the proper *.nii.gz files (0130.nii.gz, 0132.nii.gz.....0494.nii.gz )

- To run just go into any folder in /Code and run "$ python main.py"

*************************

- Numpy - Numeric Python: http://new.scipy.org/download.html
- Scipy - Open Source Library of Scientific Tools: http://www.scipy.org/
- Xlrd -  Read from Microsoft Excel: https://pypi.python.org/pypi/xlrd/0.8.0
- Xlwt -  Write to Microsoft Excel: https://pypi.python.org/pypi/xlwt
- Xlutils -  Utilities for working with Microsoft Excel: https://pypi.python.org/pypi/xlutils
- pyOSC -  OSC communication: https://trac.v2.nl/wiki/pyOSC
- Scikit-Learn - Machine Learning : http://scikit-learn.org/stable/
- Nibabel - Neuro Imaging : http://nipy.sourceforge.net/nibabel/
